# Your building pieces go here

Create one folder per piece (PascalCase name, e.g. `Woodwall`).

Each folder needs:
- `piece.ini` (use `New-Piece.bat` or copy from another piece)
- `SM_<PieceId>.uasset` (authored in Unreal)
- Material instances / textures as needed
- `T_Icon_<PieceId>.uasset` (optional menu icon)

Then open **RSDW Custom Builds Manager** and click **Build All & Install**.
